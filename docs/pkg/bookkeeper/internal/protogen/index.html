func RegisterBookkeeperHandlerClient ¶ 

func RegisterBookkeeperHandlerClient(ctx context.Context, mux *runtime.ServeMux, client BookkeeperClient) error

RegisterBookkeeperHandlerClient registers the http handlers for service Bookkeeper to "mux". The handlers forward requests to the grpc endpoint over the given implementation of "BookkeeperClient". Note: the gRPC framework executes interceptors within the gRPC handler. If the passed in "BookkeeperClient" doesn't go through the normal gRPC flow (creating a gRPC client etc.) then it will be up to the passed in "BookkeeperClient" to call the correct interceptors.

RegisterBookkeeperHandlerServer registers the http handlers for service Bookkeeper to "mux". UnaryRPC :call BookkeeperServer directly.
